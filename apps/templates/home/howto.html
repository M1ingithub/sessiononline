how to join session communities

guide for Session user.

1. download Session at:
https://getsession.org/
or:
https://github.com/session-foundation
or:
https://f-droid.org/en/packages/network.loki.messenger.fdroid/
or:
https://play.google.com/store/apps/details?id=network.loki.messenger

2. start your Session client at your phone or desktop.

3. click "create account" to have new SessionID, a cryptographic identifier unique to you
this ID servers as an 'account'. for example, if you get banned in community, you need new ID.

4. pick a username.

5. save recovery password which will enable you to recover this SessionID later if you lose access.

6. you are ready to roll! join a community by clicking the green "+" -sign at the bottom.
then copy Community URL at our website and paste it in the field "Enter community URL"


how to add session communities

guide for Session community host.

if you want to add an existing Session community, please use the add community page.

if you don't have Session community but want one and want it listed here, follow these
steps:

1.
rent a server. you need 2GB of RAM and two processor cores. this costs around
10$ / month. i rent mine at:
https://www.hetzner.com
use Ubuntu 22.04 OS

2.
create a SSH connection to the server. for this i use PuTTy:
https://putty.org/index.html

3.
follow this github guide to install SOGS on the server.
this installation should contain everything needed to run a session community:
https://github.com/session-foundation/session-pysogs/

4.
you should now be able to create your Session communities.
use "sogs --help" -command on your server.

5.
copy your Session community URL and paste it on "add community" -page.
you need to create an account and login to be able to add a Session community on our
website.


rules for added communities

all communities added to the list must act in accordance to Session ToS:
https://getsession.org/terms-of-service

please read the section 5.2 about Content Policy:
Violent Behaviour, Violent Extremism and Terrorism, Child Sexual Exploitation,
Extremely Violence and/or Graphic Content, Harmful and/or Unlawful Content
